# Timestory

"타임스토리"는 시간과 이야기를 결합한 용어입니다.
달력 앱으로써 시간에 관련된 다양한 이야기와 일정을 기록하고 관리할 수 있음을 의미합니다.
"시간의 이야기"라는 개념을 담고있어, 사용자들이 달력을 통해 일어난 일들을 기록하고 나만의 이야기를 만들어 나갈 수 있다는 의미가 담겨 있습니다.
